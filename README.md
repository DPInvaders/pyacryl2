# PyAcryl2

[Acryl](https://acrylplatform.com/ "Acryl Platform") Node API client

## Installation

From PyPI:

```bash
pip install pyacryl2
```

Or from source:

```bash
python setup.py install
```


## What's included

- [X] Node API client
	- [X] Sync
	- [X] Async
	
- [X] Generators and validators
	- [X] Address data
	- [X] Transactions data
